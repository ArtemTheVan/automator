== Назначение и область применения

*Automator* предназначен для:

* эмуляции пользовательского ввода (клавиатуры и мыши) от имени стороннего приложения;
* автоматизированного захвата экрана и регионов;
* визуального анализа экрана с помощью OpenCV (поиск трея, текстовых регионов,
  индикатора раскладки клавиатуры);
* распознавания текста (OCR) на базе Tesseract;
* выполнения сценариев автоматизации, написанных на Python, в том числе через GUI-редактор.

Целевые сценарии: тестирование пользовательских интерфейсов, автоматизация повторяющихся
операций оператора, мониторинг состояния системного трея, обучающие демонстрации.

Целевая платформа — Windows 10 и новее. Сборка ориентирована на toolchain MinGW-w64
(MSYS2 `mingw64`).


== Глоссарий

[cols="1,4", options="header"]
|===
| Термин | Определение

| Сценарий       | Python-скрипт, выполняемый через `wrapper_automator.py`.
| Регион         | Прямоугольная область экрана `(x, y, width, height)`.
| OCR            | Optical Character Recognition — распознавание текста на изображении (Tesseract).
| Трей           | Системная область уведомлений Windows.
| PSM            | Tesseract page segmentation mode.
| HKL            | Handle to a Keyboard Layout (WinAPI).
| MSYS2          | Дистрибутив POSIX-совместимой среды для Windows; источник MinGW-w64.
|===

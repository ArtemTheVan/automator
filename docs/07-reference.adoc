== Платформа и ограничения

* Целевая ОС — *Windows 10* и новее (`_WIN32_WINNT=0x0A00`).
* Сборка ориентирована на MinGW-w64; MSVC не тестировался.
* Функции OpenCV/OCR опциональны; при их отсутствии библиотека собирается без них и
  предоставляет только эмуляцию ввода и захват экрана.
* `simulate_keystroke` поддерживает базовое ASCII-подмножество; для UTF-8 используйте
  Python-обёртку или расширьте таблицу VK-кодов.
* OCR вызывает `tesseract.exe` через `system()` и ожидает фиксированный путь установки
  `C:\Program Files\Tesseract-OCR\tesseract.exe`.
* GUI собирается со статической линковкой `libgcc`/`libstdc++` для упрощения дистрибуции.


== Сборка документа

Документ написан в формате AsciiDoc, разбит на тематические файлы и подключается через
директивы `include::`. Диаграммы PlantUML рендерятся расширением `asciidoctor-diagram`.

[source,bash]
----
# Установить инструменты
gem install asciidoctor asciidoctor-diagram

# Сгенерировать единый HTML с диаграммами
asciidoctor -r asciidoctor-diagram docs/project.adoc

# Или PDF
asciidoctor-pdf -r asciidoctor-diagram docs/project.adoc
----

Для рендеринга PlantUML требуется установленный `plantuml.jar` или сетевой сервис
PlantUML.

При необходимости каждый файл `NN-*.adoc` можно собрать отдельно, добавив в его начало
заголовок документа и нужные атрибуты:

[source,asciidoc]
----
= Заголовок раздела
:toc:
:source-highlighter: rouge
:plantuml-format: svg

include::02-quickstart.adoc[]
----

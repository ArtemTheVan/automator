== Аннотация

Документ описывает архитектуру, состав и программную документацию проекта *Automator* —
кроссязыкового инструмента автоматизации действий пользователя в Windows. Проект состоит из
нативной C/C++ библиотеки `libautomator`, консольного приложения, графического Qt-редактора
сценариев и Python-обёртки. Документ ориентирован на разработчиков, которым необходимо
понимать внутреннее устройство проекта, расширять его или интегрировать в собственные
решения.

В разделах документа приведены диаграммы в нотации UML (формат PlantUML): компонентная,
структуры пакетов, классов, последовательностей и развёртывания.

Документ разбит на тематические файлы для удобства чтения:

* <<_назначение_и_область_применения,Обзор и глоссарий>> — `01-overview.adoc`
* <<_быстрый_старт,Быстрый старт>> — `02-quickstart.adoc`
* <<_архитектура,Архитектура>> — `03-architecture.adoc`
* <<_описание_программных_модулей,Описание модулей>> — `04-modules.adoc`
* <<_сценарии_работы,Сценарии работы (sequence-диаграммы)>> — `05-sequences.adoc`
* <<_сборка_проекта,Сборка и развёртывание>> — `06-build-deploy.adoc`
* <<_структура_репозитория,Справочник: структура, ограничения, сборка документа>> — `07-reference.adoc`
